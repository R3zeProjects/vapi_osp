# Схема версий

В проекте используется следующая нумерация:

| Часть версии | Значение | Пример |
|--------------|----------|--------|
| **0.** (первая цифра после мажора) | Вышедшая версия (релиз) | 0.1.0, 0.2.0 |
| **0.0** | Бета | 0.0.1, 0.0.1b1 |
| **0.0.1** | Альфа | 0.0.1, 0.0.1b1 |
| **b** (суффикс) | Правка (patch/fix) | b1, b2 → 0.0.1b1, 0.0.1b2 |

**Текущая версия API:** 0.0.1-alpha (первая альфа).

Цепочка: **0.0.1-alpha** (текущая, первая альфа) → **0.0.1b1** (правка) → **0.1.0** (первый стабильный).
