# UI: UIManager, widgets, layout, themes

UI module: widget tree, layout, input, render. Headers: ui/ui_manager.hpp, ui/widget.hpp, ui/panel.hpp, ui/button.hpp, ui/label.hpp, etc.

## UIManager

Manages root widget, font, input, render batch, theme. **init(screenW, screenH)**, **resize(screenW, screenH)**. **root()** — root widget. **update(dt)**, **buildBatch()**, **batch()**. Access via Application::uiManager().

## Widget

Base class: hierarchy, geometry, style, focus, visibility. **addChild**, **setSizePolicy**, **setPadding**, **setMargin**, **style()**, **id()**, **name()**.

## Containers

**Panel**, **HBox** / **VBox** — horizontal/vertical layout. SizePolicy (fixed, fill, percent, fit).

## Widgets

Button, Label, Checkbox, Toggle, Slider, NumberInput, InputField, TextArea, Dropdown, RadioButton/RadioGroup, ProgressBar, TabBar, Splitter, ScrollView, Table, TreeView, Dialog, ContextMenu, ColorPicker, Toast, Toolbar, StatusBar.

## Themes

**Theme** — styles by widget name/type; **UIManager::setTheme()**.
