# Оптимальные UI с отображением для коммерческого использования

В этом репозитории **ImGui не включён**; отрисовка — через собственный vapi_ui + vapi_font или сторонние библиотеки по желанию.

Краткий обзор UI-решений с рендерингом, которые **можно использовать в коммерции** (лицензии, совместимые с проприетарным кодом без обязательного раскрытия исходников).

---

## Рекомендуемый вариант: Dear ImGui + Vulkan

**Лицензия:** MIT — свободное коммерческое использование, модификация, распространение без раскрытия кода.

**Плюсы:**
- Очень распространён в индустрии (инструменты, дебаг-UI, редакторы).
- Официальные примеры под Vulkan (и OpenGL, DirectX, Metal).
- Минимальные зависимости, один заголовочный подход (header-friendly).
- Быстрая отрисовка, immediate mode — хорошо стыкуется с вашим GLFW + Vulkan.
- Нет абонентской платы и роялти.

**Интеграция:**  
Используйте бэкенд [imgui Vulkan](https://github.com/ocornut/imgui/blob/master/docs/BACKENDS.md) (например, `imgui_impl_vulkan.cpp` + `imgui_impl_glfw.cpp` из репозитория ImGui). Ваш рендер (RenderContext, swapchain) передаётся в ImGui для отрисовки в тот же pass или отдельным draw.

**Ссылки:** [dearimgui.com](https://www.dearimgui.com/), [GitHub ocornut/imgui](https://github.com/ocornut/imgui).

## Другие варианты с коммерчески дружественной лицензией

| Решение        | Лицензия | Рендеринг        | Коммерция |
|----------------|----------|-------------------|-----------|
| **OctaneGUI**  | MIT      | Агностик (Vulkan/OpenGL/D3D) | Да, без ограничений |
| **VkUI**       | MIT      | Vulkan, HTML/CSS | Да |
| **Split GUI**  | MIT      | Vulkan + GLFW    | Да |
| **libwui**     | Open-source | Свой рендер   | Уточнять лицензию |
| **WonderGUI**  | GPL + коммерческая | Свой рендер | Да по платной лицензии |
| **Slint**      | GPLv3 / платная | Skia/GPU        | Да по платной лицензии (royalty-free) |

---

## Что не подходит без доработок

- **shgui** — GPL-3.0: коммерция возможна только с выполнением GPL (раскрытие исходников производных работ).
- **Brisk** — GPL-2.0: те же ограничения для проприетарного продукта.

## Итог для вашего стека (Vulkan + GLFW)

**Оптимальный выбор:** **Dear ImGui** с бэкендами Vulkan и GLFW.  
MIT, бесплатно для коммерции, хорошо документирован, легко встраивается в существующий цикл рендера (в т.ч. после удаления кастомного UI).

При необходимости UI в стиле HTML/CSS поверх Vulkan и тоже с MIT — можно рассмотреть **VkUI**.
